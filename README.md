 ### Clone repo 
 `git clone https://github.com/cloudsoftwareoff/maali`
 ### cd to the folder
 `cd maali`

 # Server Setup
 ### install dependencies
 `npm install`
 ## run server
 ### open server folder
 `cd server`
 ## Start server
 `npx nodemon` or `node index.js`

 # Client Setup
 ## install Reactjs
 `npm install -g create-react-app`
 ### open folder
 `cd client`
 ### install dependencies
 `npm install`
 ## Start Client
 `npm start`
